simpons_test_with_context022826.csv contains the entire OJ Simpson transcript with the test sentences denoted by "TEST" in the "role" column.  The Anchor_Date_list column provides any applicalbe anchor dates for the dates in the test sentence in the order they are found in the sentence.  The Sent_GL_Normalized provides the list of applicable dates found in the test sentences at the sentence-level and the Full_GL_Normalized_Date provides the list of dates found in the test sentence at the document level in the order the dates are found in the sentence.
